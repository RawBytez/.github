## Ώθπυ What Is RawBytez?

## ΏθγΑ Project Showcase έΑΦ The Core of RawBytez

The **Project Showcase** is what makes RawBytez different from every other club.

> ΏθΦΣ **Rule #1:** You can submit **as many projects as you want.** There is no cap.  
> ΏθΦΣ **Rule #2:** There is no "too simple" or "too ambitious." All projects are valid.  
> ΏθΦΣ **Rule #3:** Projects at any stage are welcome έΑΦ idea, prototype, or fully shipped.

### ΏθΥΓ Project Submission Format

To add your project to the RawBytez showcase, open a PR or submit via the club form with:

```yaml
Project Name: "Something cool"
Builder(s): ["Your Name", "Teammate (optional)"]
Grade: 9
Description: "One-liner about what this does"
Tech Stack: ["Next.js", "Python", "whatever you used"]
Status: "In Progress / Live / Prototype"
Demo Link: "https://yourproject.com (or N/A)"
Repo: "https://github.com/you/repo (or private)"
Category: "Web App / Game / AI / Tool / CLI / Hardware / Other"
```
