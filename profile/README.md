## ЁЯзм What Is RawBytez?

> **RawBytez** is EHIS's official IT & innovation club тАФ an open, no-gatekeeping space where students at any skill level can **build anything, ship anything, and show everything**.

No prerequisites. No skill minimums. Just raw curiosity, a keyboard, and a drive to create.

Whether you're building your first HTML page or deploying a full-stack AI product тАФ RawBytez is your home base. We don't just learn tech here. We **make things with it**.

## ЁЯЪА Project Showcase тАФ The Core of RawBytez

The **Project Showcase** is what makes RawBytez different from every other club.

> ЁЯФС **Rule #1:** You can submit **as many projects as you want.** There is no cap.  
> ЁЯФС **Rule #2:** There is no "too simple" or "too ambitious." All projects are valid.  
> ЁЯФС **Rule #3:** Projects at any stage are welcome тАФ idea, prototype, or fully shipped.

### ЁЯУВ Project Submission Format

To add your project to the RawBytez showcase, open a PR or submit via the club form with:

```yaml
Project Name: "Something cool"
Builder(s): ["Your Name", "Teammate (optional)"]
Grade: 9
Description: "One-liner about what this does"
Tech Stack: ["Next.js", "Python", "whatever you used"]
Status: "In Progress / Live / Prototype"
Demo Link: "https://yourproject.com (or N/A)"
Repo: "https://github.com/you/repo (or private)"
Category: "Web App / Game / AI / Tool / CLI / Hardware / Other"
```
